# AIの使用レポート

## 使用したサービス
- Claude Code

## 使用方法
- `調査` → `計画` → `実装` -> `PRレビュー`の全ての開発ステップで利用した

### 調査
- 主にGithub APIの仕様についての調査で利用
  - 調査内容をレポートさせ、人間による一次情報の再調査を実施
  - 初期段階では旧バージョンを前提に調査が進んでいたため、最新版をもとに再調査を実行するように人間による指示を行った

### 計画
- 人間がissue化したタスク単位で実装方針をplanモードで作成
- 人間によるレビューで、React/Next.jsのベストプラクティスを無視するなどの想定と異なるプランが作成された場合に修正指示を行い、仕様を完全に固めてから実装段階に移行

### 実装
- プランファイルをもとに実装をさせた
- プランファイルをもとにテストコードを実装し、テスト内容が意図通りであることを人間がレビューしてから、アプリケーションロジックの実装を進めた

### PRレビュー
- デフォルトの`/review`コマンドを使用し、PRをレビューしてもらった
- 指摘内容を人間が再レビューし、妥当だと判断したコメントについて修正対応を行った
- 適宜、[react-best-practices skill](https://vercel.com/blog/introducing-react-best-practices)を使用して改善ポイントを指摘してもらい、同様に人間によるレビューを行った後に妥当な内容の対応を行った
